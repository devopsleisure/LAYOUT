## ◉ &nbsp;Deployment Guide

### ⚡ Option 1 — GitHub Pages (Recommended)

```bash
# 1. Clone or fork this repository
git clone https://github.com/your-username/sattar-portfolio.git

# 2. Push to GitHub
git add .
git commit -m "🚀 Initial portfolio deploy"
git push origin main
```

Then: **Settings → Pages → Source → `main` / `root` → Save**

Your site will be live at:
```
https://your-username.github.io/sattar-portfolio/
```

---

### 🌐 Option 2 — Netlify (Drag & Drop, 30 seconds)

1. Go to **[netlify.com](https://www.netlify.com)** → Log in
2. Drag the entire `sattar-portfolio/` folder onto the deploy zone
3. Done — live URL generated instantly

---

### 💻 Option 3 — Run Locally

No server, no dependencies, no build step needed:

```bash
# Just open the file directly
open index.html        # macOS
start index.html       # Windows
xdg-open index.html   # Linux
```

## ◇ &nbsp;Customization

### 🖼️ Add Your Profile Photo

Drop your photo as `assets/images/profile.jpg`, then in `index.html` replace:

```html
<!-- BEFORE -->
<div class="profile-img-placeholder">SA</div>

<!-- AFTER -->
<img src="assets/images/profile.jpg" alt="Portrait of Sattar Ahmed" class="profile-img"/>
```

---

### 🎨 Change Colors

All design tokens live at the top of `css/style.css`:

```css
:root {
  --jade:        #0B3D2E;  /* Primary background */
  --champagne:   #E8C97A;  /* Gold accent        */
  --emerald:     #2ECC71;  /* Live / success      */
  --teal:        #1ABC9C;  /* Subtitle accent     */
  --white:       #F7F3EC;  /* Body text           */
}
```

---

### 📧 Enable Silent Email Sending (EmailJS)

The contact form currently opens Gmail compose in a new tab.
To send emails silently in the background:

1. Sign up at **[emailjs.com](https://www.emailjs.com)** (free tier)
2. Add Gmail service → copy **Service ID**
3. Create email template → copy **Template ID**
4. Copy your **Public Key** from Account settings
5. Add to `index.html` before `script.js`:

```html
<script src="https://cdn.jsdelivr.net/npm/@emailjs/browser@4/dist/email.min.js"></script>
```

6. Replace the `window.open(gmailUrl...)` block in `js/script.js` with:

```js
emailjs.init('YOUR_PUBLIC_KEY');
emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', {
  from_name: name,
  from_email: email,
  subject: subject,
  message: message,
  to_email: 'sattarahmedofficial1@gmail.com'
});
```
